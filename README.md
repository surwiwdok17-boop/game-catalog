# game-catalog
Каталог Ігор (Game Catalog)
Це консольний застосунок для обліку вашої ігрової колекції. Програма дозволяє користувачам ефективно керувати списком ігор, відстежувати прогрес проходження та аналізувати середній рейтинг колекції.

 Про проєкт
Метою проєкту «Каталог ігор» є розв'язання проблеми хаотичного обліку ігрової колекції користувача. Додаток надає централізований інтерфейс для фіксації та перегляду ігор, їхніх характеристик та оцінок. Система автоматизує рутинні операції — від сортування ігор за рейтингом до формування аналітичних звітів, перетворюючи звичайний список на структуровану базу даних, що значно спрощує навігацію власною бібліотекою.

 Можливості
Додавання: Інтерактивне введення даних з автоматичною валідацією.
Аналітика: Розрахунок середнього рейтингу та виведення статистики.
Сортування: Швидке впорядкування колекції за рейтингом.
Фільтрація: Пошук ігор за статусом проходження.
Надійність: Використання формату JSON для збереження даних та автоматична обробка помилок вводу.

## 🚀 Інструкція із запуску

 **Клонуйте репозиторій:**
   ```bash
   git clone [https://github.com/surwiwdok17-boop/game-catalog.git](https://github.com/surwiwdok17-boop/game-catalog.git)
Запуск через командний рядок:
Перейдіть у папку з проєктом і виконайте:

Bash
python catalog.py
Швидкий запуск (Windows):
Просто двічі клацніть на файл run_catalog.bat.

 Структура проєкту
catalog.py — основний код програми.
games_catalog.json — база даних (створюється автоматично).
run_catalog.bat — скрипт для швидкого запуску в Windows.
test_catalog.py — модуль автоматизованого тестування.
README.md — документація проєкту.
