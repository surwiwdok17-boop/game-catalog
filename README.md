Каталог Ігор (Game Catalog)
Проєкт «Каталог ігор» — це консольна система управління персональною бібліотекою відеоігор, розроблена для систематизації та зручного відстеження ігрового прогресу. Програма дозволяє користувачеві не просто зберігати список ігор, а й аналізувати власні вподобання через статистику, фільтрацію та гнучку систему оцінювання.

Про проєкт
Головна мета проєкту — забезпечити швидкий доступ до даних про платформу, статус проходження та рейтинг кожного тайтла в єдиному цифровому середовищі. Система автоматизує рутинні операції — від сортування ігор за рейтингом до формування аналітичних звітів.

Можливості
Додавання: Інтерактивне введення даних з автоматичною валідацією.
Аналітика: Розрахунок середнього рейтингу та виведення статистики.
Сортування: Швидке впорядкування колекції за рейтингом.
Фільтрація: Пошук ігор за статусом проходження (граю, пройдено, відкладено).
Надійність: Використання формату JSON для збереження даних та автоматична обробка помилок вводу.

Інструкція із запуску
Вимоги
Встановлений Python 3.x.

Запуск
Клонуйте репозиторій:
git clone https://github.com/surwiwdok17-boop/game-catalog.git

Запуск через командний рядок:
Перейдіть у папку з проєктом і виконайте команду:
python catalog.py

Швидкий запуск (Windows):
Просто двічі клацніть на файл run_catalog.bat.

Структура проєкту
catalog.py — основний код програми.
games_catalog.json — база даних (створюється автоматично).
run_catalog.bat — скрипт для швидкого запуску.
test_catalog.py — модуль автоматизованого тестування.
README.md — документація проєкту.

🛠 Технології
Мова: Python 3.
Формат даних: JSON.
Бібліотеки: json, os, sys.

🧪 Тестування
Проєкт включає модуль автоматизованого тестування для перевірки ключових алгоритмів. Щоб запустити тести, виконайте:
python test_catalog.py
